# React + Vite Slider

## Installation

### Download Node

download node if you havn't already

```bash
https://nodejs.org/en/download/current
```

Use the node package manager [npm](https://nodejs.org/en/download/current) to install dependencies.

```bash
npm i
```

## Usage

```React
npm run dev
```
